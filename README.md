### Asymptotic classes
	
first class: part of the two-particle reducible vertex depending on one bosonic frequency
	
$\mathcal{K}_1^r$ \cite{gievers2022multiloop,ge2024real}, $K_{1,c}$ \cite{ritter2022benchmark}, $\mathcal{K}^{(1)X}$ \cite{fraboulet2022single}, $\mathcal{K}_{1,c}$ \cite{Wentzell2020}
